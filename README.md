# Sommelier

## Requrirments 
* [Docker](https://www.docker.com/)
* [Leiningen](https://leiningen.org/)
* [Auth0](https://auth0.com/)

## Installation

### Local

```bash
// start
./gradlew build run

// stop
./gradle stop
```

### AWS

```bash
// start
./gradlew build publish deploy
```
